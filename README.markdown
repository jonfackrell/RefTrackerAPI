# RefTrackerAPI #

RefTrackerAPI is a full implementation of the RefTracker API for RefPages.

# Features #
* Lightweight
* Compatible with all major browsers (Tested on Firefox 3.6, Firefox 4.0, Firefox 5.0, Chrome 9, Opera 11, IE7, IE8, IE9)
